0. Yêu cầu: Cài đặt Visual Studio, DevExpress Components cho Winform (từ phiên bản 18.1 trở về sau) và SQL Server Management từ bản 2012 về sau.

1. Vào folder Queries/CreateDatabase.sql và execute file để thực hiện tạo database.

2. Vào folder Queries/StoreProcedure.sql và execute file để thực hiện tạo store procedures.

3. Mở solution, trong file DBConnection (thuộc project DAO), sửa lại đường dẫn connectionString phù hợp.

4. Thực hiện các test theo mô tả.

5. Lưu ý: Với project setup, vì file .mdf được đóng gói bởi SSMS phiên bản 2017 nên muốn thực hiện cần cài đặt SQL Server Management từ bản 2017 về sau.
